The github repository includes:

- A "data" folder containing a file "communities.names" describing the data used for this project.
  The actual data can be found here : https://archive.ics.uci.edu/ml/machine-learning-databases/communities/

- A "saved array" folder containing all the arrays generated during training: these arrays can be loaded back as variables in the python notebook

- A project.ipynb file: the main notebook containing the GD and ZO algorithms. It also includes plots  used for the work study.
